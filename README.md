# CAD Designs

A collection of robotics CAD designs, PCB schematics, and Arduino code for various projects.

## Projects

### Hexapod Robots
- **Aecerts Hexapod V1** - Full hexapod robot with custom gearboxes, RC controller, and Arduino code
- **Sci-Fi Hexapod Robot** - Stylized hexapod design
- **SIXpack Robot** - Six-legged FPV walking robot

### Quadruped Robots
- **Q1 Mini** - Compact quadruped robot by Jason Workshop
- **FOURky** - Four-legged robot with ESP control
- **ESPiderman** - ESP-based spider robot

### Other Projects
- **DIY Stream Deck** - Custom stream deck with 3D printing and Arduino
- **Drone** - Custom drone PCB designs
- **TurtleBot** - Gazebo simulation files for TurtleBot3

## Contents

- `/Aecerts_Hexapod_V1-main` - Complete hexapod project (STLs, code, PCB, RC controller)
- `/Drone` - KiCad PCB files for drone
- `/Gerber File` - PCB manufacturing files for RC transmitter
- `/ESPidermanCults3D` - Spider robot STLs and firmware
- `/FOURkyCults3D` - Four-legged robot files
- `/Q1 mini Quadruped Robot` - Quadruped STLs and images
- `/Turtle` - TurtleBot3 Gazebo models and plugins
- `/diy-stream-deck-main` - Stream deck Arduino code and CAD files

## File Types

- `.stl` / `.STL` - 3D printable models
- `.step` - CAD source files
- `.ino` - Arduino sketches
- `.kicad_*` - KiCad PCB/schematic files
- `.gbr` - Gerber manufacturing files
